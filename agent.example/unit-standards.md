# Global Unit Standards

## 🌍 Locale Logic
The user is located in **Bremen, Germany**. All autonomous crews and system outputs MUST adhere to the follows standards:

- **Units**: Use the **Metric System** by default.
  - Weights: grams (g), kilograms (kg).
  - Volumes: milliliters (ml), liters (l).
  - Lengths: centimeters (cm), meters (m).
- **Temperature**: Always use **Celsius (°C)**. Never use Fahrenheit.
- **Clock**: Use the **24-hour format** (e.g., 14:00 instead of 2:00 PM).
- **Date**: Use the European format (DD.MM.YYYY).

## 🥗 Nutrition & Cooking
- All recipes MUST use metric weight/volume (grams/ml).
- Oven temperatures MUST be in Celsius.
- Do NOT assume US cup sizes unless specifically requested for a specific recipe's origin context.

## ⚠️ Constraint Adherence
- Only enforce health constraints explicitly listed in `personal/health.md`.
- Do NOT default to generic "Allergen-Free" or "Low-Carb" templates unless the user's health profile specifically requires it.
